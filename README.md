## 功能介紹

這個Line Bot可以幫助你快速決定在交大附近要吃什麼。你只需要告訴它你想吃的時段和地點，它就會隨機推薦餐廳給你。

## 使用說明

1. **啟動機器人**
   當你第一次使用機器人時，會收到歡迎訊息：
   ```
   輸入其中一項就開始吧：
   早餐/午餐/晚餐/宵夜/飲料甜點
   ```

2. **選擇用餐時段**
   輸入你想要的用餐時段，例如 `早餐`、`午餐`、`晚餐`、`宵夜` 或 `飲料甜點`。

3. **選擇用餐地點**
   機器人會提示你選擇地點：
   ```
   尼想在哪吃(੭ ᐕ)੭？ 一餐/二餐/女二/研三/清夜/隨便
   ```

4. **獲得推薦**
   根據你的選擇，機器人會隨機推薦一個餐廳。例如：
   ```
   就決定 4 尼ㄌ！二餐 - 茶壜 - 雙Q百香綠 (๑ơ ₃ ơ)♥
   ```
